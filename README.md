# Raices Flooring - Client Website

## Overview
This repository contains the source code for a responsive, client-facing website that was designed, built, and deployed as a complete end-to-end project. The focus was on usability, clean design, and a straightforward deployment workflow.

A live version of the site is publicly available.

## Live Demo
**Website:** https://YOUR-LIVE-URL-HERE](https://raicesflooring.com/

## Tech Stack
- Frontend: React, JavaScript, HTML, CSS  
- Styling: Tailwind CSS  
- Form Handling: Formspree  
- Hosting: Vercel / Netlify / GitHub Pages  
- Version Control: Git, GitHub

## Features
- Responsive layout for desktop and mobile
- Clean navigation and accessible UI components
- Contact form with client-side validation
- Deployed and publicly accessible

## Contact Form
The contact form uses Formspree for handling submissions without a custom backend.  
No API keys or sensitive credentials are stored in the repository.  
Spam protection and submission limits are managed through Formspree settings.
